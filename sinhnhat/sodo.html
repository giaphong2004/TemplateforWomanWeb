Sơ đồ Kiến trúc Hệ thống

Mô hình kiến trúc 3 tầng (3-Tier Architecture)

Presentation (Frontend)

Giao diện người dùng

  • Angular 19: Framework chính cho giao diện.
  • TypeScript: Ngôn ngữ lập trình an toàn kiểu.
  • Responsive UI: Giao diện đáp ứng mọi thiết bị.
  • Component-based: Kiến trúc dựa trên thành phần.

HTTP/HTTPS

Application (Backend)

Logic nghiệp vụ

  • Node.js/Express: Nền tảng xử lý logic.
  • Sequelize ORM: Ánh xạ đối tượng-quan hệ.
  • JWT Authentication: Cơ chế xác thực an toàn.
  • RESTful APIs: Giao tiếp với Frontend.

ORM/SQL

Data Access (Database)

Lưu trữ dữ liệu

  • MySQL Database: Hệ quản trị CSDL quan hệ.
  • Relational Model: Lưu trữ theo mô hình quan hệ.
  • ACID Properties: Đảm bảo tính toàn vẹn giao dịch.
  • Data Integrity: Đảm bảo tính nhất quán dữ liệu.
